[ 購物流程 ]

還未成為本網站會員者,請填寫完整資料並且加入會員或以FB帳號登入，如以成為會員者,請於結帳時登入會員即可

[ 運費計算 ]

  • 1. 7-11取貨付款 60元
  • 2. 黑貓宅配到府 80元
  • 3. 門市自取免運費

[ 配送所需時間 ]

1. 黑貓宅急便：寄出後大約一個工作天會宅配到府，台灣本島、澎湖部分地區、金門部分地區 (大金門、小金門)、小琉球、馬祖全部 地區、綠島。以下區域除外：蘭嶼，澎湖：望安鄉、七美鄉、虎井島、桶盤島、大倉嶼、員貝嶼、鳥嶼、吉貝嶼　金門：烏坵鄉(大膽島、二膽島)

2. 7-11取貨付款：寄出大約2-3天到指定門市，會以簡訊通知，請收到簡訊後再前往取貨，請參考7-11門市據點＊提醒：單筆訂購二雙靴鞋以上或內含長靴，依超商代收的規定，無法使用其服務。請選擇宅配到府服務

3. 門市自取：約1~2個工作天會送達指定門市，商品出貨會EMAIL通知，商品到店會發送簡訊，請收到簡訊後再前往取貨

備註：1. 若遇國定假日(例假日)，因金融機構系統假日無法立即入帳之原因，CHIC CHIC GIRL將以確實核對到帳款日來計算配送時間。 (如遇不可抗力之因素，例如：颱風‧地震及水災….等，將視當時狀況處理)

2. 購買正價商品滿$4000 即可以享免運優惠，特價商品除外。

[ 付款方式 ]

信用卡線上刷卡：凡各家銀行的VISA，MASTER，JCB 與中國銀聯信用卡皆可使用 。

信用卡分期: 僅提供中國信託、國泰世華銀行的3期、6期零利率付款

超商代碼付款：全省 7-11、全家、OK、萊爾富等便利商店都可繳款，依您選擇的便利商店後，系統就會產出一組"超商"的繳費代碼，要記下來或直接到您的信箱收取訂單確認信並列印訂單，然後再到超商門市機器輸入代碼印出繳費單，並至櫃台繳費即可。門市操作步驟 : 超商機器輸入--> 繳費 --> 代碼繳費--> 輸入代碼 --> 印出單據 --> 櫃台付款

ATM虛擬帳戶匯款：完成結帳時將取得當次訂單的專屬虛擬帳號訂購完成後，系統會產生一組專屬於這筆訂單的十六位數虛擬轉帳帳號。取得帳號後隔天 (23:59 以前 )完成匯款即可，完成匯款後不會再要求提供帳號等資料。
